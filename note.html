  1. CSS Layout: Layout is desined by HTML and CSS. We will use Float for now. Grid and Flex will learn during bootstrap.
  2. Create a header division: assign class and apply properties background color to view and must have to give heigth or content to show on page. Width 100% will automatically take as block level element. Width only applicable if want to minimize the width
    Height versus min-height:
    • if height not set, it automatically take as content required
    • If heigth set lower than content required space then background will show only as setted heigh even if content require more. Use of height is not profesional.
    • minimum-height will always show a background as minimum height set but atomatically increase as content required. If we want to height then minimum-height is best... mostly avaoid any height
    • maximum-height will allow only upto a setted height
  3. Wrappe concept: used when two div want to place same line. also known as container. We have to assign main and side division under wrapper division.

  7. Now a problem will arise, that next content will go in that 5% gap. So we have to use any of those option
    • overflow: hidden in immidiate paranet class. Best option, becasue it will use full area of this division. and clear option will only restict next element go up.
    • clear:both/left/right kit can be used in immidiate nex div to not use that gap.
      • clear: left will slove only left side
      • clear: right will slove only right side
      • clear: both will slove only both side
